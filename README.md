# 🎙️ ASR Error Correction for L2 Learners

This project improves speech recognition output for non-native English speakers.

## 🚀 Features
- Whisper ASR for transcription
- NLP-based text correction
- Streamlit web interface

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
## 📂 Dataset

The L2_ARCTIC dataset is used in this project.

Due to size limitations, the dataset is not included in this repository.
